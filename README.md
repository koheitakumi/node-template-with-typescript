## Purpose

_Boiler Template for NodeJS project with TypeScript_.

We can start a new project with this clean template with TypeScript that keep the code better.

## Install

1. Clone this repository

   ```bash
   git clone
   ```

1. Rename the project name

   - Project Directory name
   - package.json
   - remove src/index.ts
   - change contents as you like
     - .vscode/settings.json
     - .editorconfig
     - .gitignore
     - .prettierrc.js
     - tsconfig.json

1. Commit these changes and push it into your Git repository.

   ```bash
   git commit -m "first commit"
   git remote add origin https://github.com/koheitakumi/node-template-with-typescript.git
   git push -u origin master
   ```
